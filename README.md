Jay Flower Mart - Static Website
---------------------------------

Contents:
- index.html
- assets/css/styles.css
- assets/images/* (6 generated sample images)
- README.md

How to view locally:
1. Unzip the package.
2. Open index.html in your browser (double-click).

How to publish to GitHub Pages quickly:
1. Create a new GitHub repository.
2. Push all files to the repo (main branch).
3. In repo Settings -> Pages, set branch to 'main' and folder '/'.
4. Wait a minute and your site will be live.
